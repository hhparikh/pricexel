Template Instalation

Step 1

Don’t have an Pagewiz account yet? Sign up for a Free account right now. If you do have one, sign in at http://cp.pagewiz.com/Login

Step 2

From the main control panel click the down arrow to the right of "Create New Landing Page". Then select "Import from file".

Step 3

Select your file from the resulting pop-up window and then choose "Upload".

Features

Adventour is a premium multi-purpose template for your landing page based on Pagewiz. It has both desktop and mobile versions.

It comes with:

  • Desktop & Mobile versions
  • CSS Animations
  • Drag & Drop
  • Layered PSD
  • 475 Icons
  • Multiple Columns
  • Key Features Section
  • Popular Places
  • Call to Action
  • Photogallery
  • Subscribe Form
  • Contact Form
  • Testimonials
